# regressionCDM

Corrected Regression Analysis For Cognitive Diagnosis Models

## Overview

This package implemented corrected regression for including covariates in cognitive diagnosis or handling distal outcome variables

## Installation

Install the development version from GitHub:

```r
# install.packages("devtools")
devtools::install_github("xpsychometrics/regressionCDM")
```

Or using `remotes`:

```r
# install.packages("remotes")
remotes::install_github("xpsychometrics/regressionCDM")
```
